
# Minimal 
minimalistic template for JS client-side app

* yarn (npm should work as well)
* babel (ES6)
* webpack 2
* webpack-dev-server


### `yarn install` or `npm install`
- install dev dependencies

### `yarn run dev`
- start dev server with hot-reload
- go to http://localhost:9000/

### `yarn run build`
- build app to /dist
